Difference between Div and Span

Div tag is the block-level element that stands for the division it takes the whole width of screen so the next element which will come in the next line whereas span tag is the inline element as it takes the space that is required for content so that the content will start from same line only.

Difference between class and IDs

  • One element can have more than one class i.e. class="head head1" but one element can have only one id i.e. id="head"
  • IDs are unique but class are not unique.
  • We can style element by both as class="box" .box{} but in id id="heading" #heading

What are self closing tags

Self closing tag is that which does not need a closing tag which means there is no separate tag. Elements that don't have any content are called empty elements. We can close empty tag is opening tag like /br. Examples of self closing tag br,hr,img and input.

How many types of headings are there ? How to decide which to use where

Heading tags are titles that are display in our webpage. There are six levels of heading tags in HTML i.e. h1,h2,h3,h4,h5,h6 h1 is main heading in webpage. h2 and h3 use to give sub headings in page h4,h5 and h6 use to give content. h1 is of highest importance and h6 is of least importance.

Difference between block and inline?

Block element covers the entire width of screen. Examples are div,p,h1 to h6,table and form etc. But inline elements cover the required width of content. Examples are a,span,img,input,label,etc.

Difference between em and strong?

The strong and em tags are used for emphasizing parts of a text. The strong tag should be used to indicate strong importance. It makes text look bold but em tag is used to emphasize the text in a different tone of voice.

What are Attributes?

HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of a HTML element type. Attributes should be written in lowercase only as they are case sensitive.It should always be applied with the start tag, we can add multiple attributes to an element by giving space between two attributes. Examples : <img src="https://www.google.com/img" alt="image"> here src and alt are attributes.

Difference between HTML Tags and Elements?

HTML tags are the starting and ending part of an HTML element. They begin with the “<” symbol and end with “>”. Whatever written inside < and > are called tags. HTML Elements enclose the contents in between the tags. They consist of some kind of structure. It generally consists of a start tag, content and end tag.
